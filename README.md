# School benchmark

Вопросы по ЕГЭ как способ оценить сильные и слабые стороны LLM. 
Это нужно для принятия обоснованного решения - какую модель выбрать для своего чат-бота.


### Для контрибьюторов

1. Используйте официальные источники: 
- https://fipi.ru/ege/demoversii-specifikacii-kodifikatory  
- https://education.yandex.ru/ege/inf  
- https://4ege.ru/matematika/76070-demoversii-ege-2026-po-matematike.html
- и т.д.

2. Банк вопросов по предмету должен содержать 100 вопросов с различным уровнем сложности в пропорциях:
- простой - 50
- средний - 30
- сложный - 20
Для удоства можете разделить на три файла.
